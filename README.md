# Shell Scripting Automation Project

## Introduction
This project automates Linux system tasks using shell scripting.

## Features
- Automatic backup
- Log cleanup
- Disk usage monitoring
- Cron job automation

## Technologies Used
- Bash Shell
- Linux
- Git & GitHub
- Cron Jobs

## How to Run
```bash
chmod +x backup.sh
./backup.sh
```

## Cron Automation
This project uses cron jobs to run scripts automatically.

## Backup System
The backup script creates backups of important files and stores them safely.
